# GARUD

GARUD is a personal AI assistant featuring voice interaction, computer vision, local system operations, and a dynamic holographic UI.

## Features
- **Voice Interaction:** Speech recognition and text-to-speech feedback.
- **Vision Capabilities:** Computer vision functionalities for analyzing visual inputs.
- **Holographic UI:** A futuristic interface for an immersive user experience.
- **Local Integration:** Performs local file system and OS operations seamlessly.

## Getting Started

1. Clone the repository
2. Install Python dependencies: `pip install -r requirements.txt`
3. Setup `.env` file with required API keys
4. Run the main application: `python main.py`

## Requirements
- Python
- Required API keys (add to `.env`)
